إليكم نص الإعلان باللغتين العربية والإنجليزية.
العربية:

سأكون مسرورًا لإعلان أنني بدأت العمل على ترجمة كتاب «Ubuntu Server Guide» إلى اللغة العربية! كما تعلمون، إنه كتابٌ رائع عن إدارة الأنظمة، مُعَدّ مباشرةً من مصدر توزيعة أوبنتو.

يبدأ الكتاب بشرح عملية التثبيت وإدارة الحزم، ثم يبيّن كيفية ضبط الاتصالات الشبكية، DM-Multipath، والإدارة عن بعد، وطرائق الاستيثاق الشبكي، ومراقبة أداء الخادم وكيفية تأمين خدماته؛ ثم يكمل إلى توضيح كيفية ضبط مختلف الخدمات، مثل أباتشي، MySQL، PostgreSQL، وتطبيقات LAMP، وخوادم الملفات والبريد والمحادثة الفورية؛ ثم ينتهي بفصولٍ عن أخذ النسخ الاحتياطية لبياناتك أنظمتك، والأنظمة الوهمية، وشبكات VPN، وغيرها من المتفرقات.

إنني سعيد لأن أشارك في هذا المشروع الرائع! آملًا أن أنجز شيئًا ملحوظًا للمجتمع التقني العربي، وأن أساعد العديد من مدراء الأنظمة.

عبد اللطيف ايمش


English:

I am glad to announce that I have started working on the translation of “Ubuntu Server Guide” to Arabic! As you know, it is an excellent book about server administration, right from the source of the Ubuntu distribution.

It starts with the installation process and package management, then it explains how to set up network communication, DM-Multipath, remote administration, network authentication methods, and monitoring the server performance and how to secure it’s services. And it proceeds to detailing how to configure different services; like Apache, MySQL and PostgreSQL, LAMP applications, File, Email and chat services. Finally it ends with chapters about backing up your systems, visualization, VPN networking, and miscellaneous.

I am glad to participate in this fantastic project. Hoping to make something remarkable in the technical community of the Arab world, and help a lot of system administrators out there.

Abd Allatif Eymsh
